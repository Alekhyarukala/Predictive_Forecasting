# 📌 Project Overview

The **UAC Predictive Intelligence System** is an AI-powered operational intelligence platform designed to forecast:

- UAC care load demand
- Operational pressure
- Discharge trends
- Capacity risk
- Future forecasting scenarios

The system combines:

- Machine Learning
- Statistical Forecasting
- Interactive Dashboards
- Risk Intelligence
- Time Series Analysis

to support data-driven operational planning.

# ⚙ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Alekhyarukala/Predictive_Forecasting.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd Predictive_Forecasting
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Streamlit Application

```bash
streamlit run streamlit_app.py
```
